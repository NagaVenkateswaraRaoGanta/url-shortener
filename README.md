# 🔗 URL Shortener

A simple full-stack URL shortener web application built using Spring Boot, MySQL, and HTML/CSS/JavaScript.

---

## 🚀 Features

- Shorten long URLs
- Redirect to original URL
- Clean and modern UI
- Copy short URL functionality

---

## 🛠️ Tech Stack

- Backend: Spring Boot (Java)
- Database: MySQL
- Frontend: HTML, CSS, JavaScript

---

## 📂 Project Structure

src/
 ├── controller
 ├── service
 ├── repository
 ├── model
 └── resources/static

---

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/your-username/url-shortener.git

2. Navigate into project:
cd urlshortener

3. Run the application:
.\mvnw.cmd spring-boot:run

4. Open browser:
http://localhost:8080

---

## 🧪 Example

Input:
https://google.com

Output:
http://localhost:8080/r/abc123
